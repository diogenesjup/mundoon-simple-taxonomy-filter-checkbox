Simples e eficaz filtros de taxonomias com checkbox.

== Description ==

O Plugin Mundoon Simple Taxonomie Filter Checkbox gera uma lista de filtros para a página archive.
Através de um painel de opções você pode selecionar quais taxonomias do post type serão exibidos e alterar os seus respectivos títulos.


== Installation ==

1 - Upload o mundoon-stfc para a pasta "/wp-content/plugins/"
2 - Ative o plugin no menu Plugins do wordpress
3 - No menu do wordpress vá para configurações e depois em Mundoon Filter
4 - Selecione as taxonomias que deseja para cada post type
5 - No template archive-{post-type}.php insira a função com o nome do post type. Exemplo: mo_stf('post-type-name');
